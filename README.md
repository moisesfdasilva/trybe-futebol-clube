# Trybe Futebol Clube

## Sobre o Projeto
Aplicação desenvolvida para apresentar informações sobre partidas e classificações em campeonatos de futebol.

## Tecnologias e Ferramentas empregadas
Node.js;
TypeScript;
Express;
Sequelize;
Chai;
Mysql; e
Docker.
